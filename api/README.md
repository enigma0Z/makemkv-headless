# MakeMKV Headless API

An API server for MakeMKV

## Usage

```
usage: mmh_api [-h] [--source SOURCE] [--log-level LOG_LEVEL] [--log-file LOG_FILE] [--port PORT] [--frontend FRONTEND] [--cors-origin CORS_ORIGIN]

options:
  -h, --help            show this help message and exit
  --source SOURCE
  --log-level LOG_LEVEL
  --log-file LOG_FILE
  --port PORT
  --frontend FRONTEND
  --cors-origin CORS_ORIGIN
```
